- **캘린더**: Google Calendar API v3

### 데이터베이스 스키마 (예정)
```sql
-- users 테이블
CREATE TABLE users (
  id UUID PRIMARY KEY,
  email VARCHAR,
  gender VARCHAR,
  age_range VARCHAR,
  occupation VARCHAR,
  style_preference VARCHAR,
  created_at TIMESTAMP
);

-- recommendations 테이블  
CREATE TABLE recommendations (
  id UUID PRIMARY KEY,
  user_id UUID REFERENCES users(id),
  weather_data JSONB,
  calendar_data JSONB,
  recommended_outfit TEXT,
  feedback INTEGER, -- -1, 0, 1
  created_at TIMESTAMP
);
```
